## 🔧 Tech Stack

| Frontend | Backend | Database | Authentication |
|----------|---------|----------|----------------|
| React.js | Node.js | MongoDB  | JWT, bcrypt    |
| CSS | Express | Mongoose | Role-based     |

---

## 🚀 Features

- 🔍 Browse properties by location, price, and type
- 🧾 Add, edit, or delete property listings (for sellers)
- 🔐 Secure login/signup with JWT authentication
- 🧑‍💼 Role-based access: Buyer / Seller / Admin
- 📸 Upload images for property listings
- 📱 Fully responsive design
